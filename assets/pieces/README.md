# Pieces

Place your custom pieces in a subdirectory of the `pieces` directory.
The following files should exists in the subdirectory:

* `king.txt`
* `queen.txt`
* `rook.txt`
* `bishop.txt`
* `knight.txt`
* `pawn.txt`

Each file should contain the pieces ASCII art in the format 7x14. Example:
```
              
      ██      
    ██████    
      ██      
     ████     
     ████     
   ████████   
```
